电路元件库手册

Logisim-evolution 电路元件库包含一组工具，允许您通过在画布区域中单击并拖动鼠标来与电路进行交互。 大多数情况下，工具旨在将特定类型的组件添加到电路中； 但一些最重要的工具，例如 Poke 工具 () 和选择工具 (），允许您以其他方式与组件交互。

通常，当选择或添加组件时，Alt-0 到 Alt-9 用于更改其 数据宽度属性。 快速使用按键可以输入更大的值，例如：ALT-1 ALT-2 代表12。

Logisim-evolution 内置库中包含的所有工具均记录在本参考材料中。
